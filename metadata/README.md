# UCC-Chain Pilot Filing Metadata

This folder contains off-chain metadata JSON files for UCC-Chain
attestations. Each file corresponds to one UCC-1 filing and contains
the inputs required to verify the on-chain attestation.

## File Naming Convention

Files are named by filing ID:
